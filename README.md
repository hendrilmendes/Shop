### Recursos:

* App feito em Flutter, disponível para iOS e Android
* Firebase para monitorar o desempenho, erros do app e autenticação
* Onesignal para envio de notificações
* Salvar Postagens
* Comentar em Posts
* Leitor de Texto
* Modo Escuro
* API Blogger v3
